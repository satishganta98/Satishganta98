Place official certification badge images in this folder using the exact filenames below:

- comptia-security-plus.png
- comptia-network-plus.png
- comptia-cysa-plus.png
- splunk-cybersecurity-defense-analyst.png
- career-essentials-system-administration.png
- comptia-csap.png
- microsoft-sc-300.png

Recommended format: PNG with transparent background.
Recommended height: ~80px to 140px.

After adding files, the certifications cards will automatically show badge images.
If a badge image is missing, the UI falls back to issuer logo, then certification name.
